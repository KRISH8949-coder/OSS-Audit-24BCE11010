## **Project Description**
This project is an **Open Source Software Audit** of the VLC Media Player. The audit focuses on verifying the software's installation, checking system-level footprints, and ensuring compliance with FOSS principles using automated shell scripts.

---

## **Shell Scripts Detail**
| File Name | Audit Function |
| :--- | :--- |
| **script1.sh** | Checks System Identity & VLC Binary Path |
| **script2.sh** | Performs Package Verification (DPKG check) |
| **script3.sh** | Audits File Permissions for Security |
| **script4.sh** | Analyzes Disk Usage & Storage Footprint |
| **script5.sh** | Generates an Interactive Final Audit Report |

---

## **How to Run**
1. Open the Linux Terminal.
2. Provide execution permissions: `chmod +x *.sh`
3. Execute the scripts: `./script1.sh` (Repeat for all 5).
